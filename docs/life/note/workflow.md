# 一种工作方式

latest version: v1.0.1

## 工作上

把一切 **我不想做(无论是因难度还是爱好)的事或是因他人安排而做的事**视为工作，逼自己每天做至少**7h**工作，每完成一个小时就记录下来，工作时段只能做工作上的task，如果这个工作我做着很开心，超过7h也没问题，否则到了就可以停手

在工作指标没达到之前，不能使用互联网进行任何娱乐，最多可以用实际性的爱好放松一下。

工作时段的休息: boring break (盯墙5~15min)，工作前听听《labor of love》并闭眼深呼吸回到状态

!!! tip "惩罚与奖励"
    - 做公益捐款: 1h→1 | 2h→3 | 3h→7 | 4h→15 | 5h→30 | 6h→60 | 7h→100
    - 在月初记账时先扣除1000作为可能的捐款库，而每周可以从捐款库取回1/4，当月结束时直接全部取出，取出的作为自由支配的部分
    - 连续一周达标可以换取一天休息，只需要完成3h的指标就可以自由行动。
    - 连续三天不达标取消本周赛博娱乐。
